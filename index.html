The Platinum Lyric Project

Heroku deployed, Python/Flask web application developed as part of a team project, integrated with a PostgreSQL database, accessed using SQLAlchemy, and a Machine Learning model to make predictions based on user input. Additionally, Plotly visualizations provide further insights into the data.

The project aims to answer the question of whether a song's lyrics can predict its success, which was defined as reaching the Billboard Hot 100 list, and exploring other possible factors that can contribute to that success.

The machine learning model used was Complement Naive Bayes, since it presented the highest accuracy and F1 scores of the models analyzed. We considered the model sufficiently accurate, considering the only features were the words in the song, however it tends to do much better at predicting the "failures" rather than the "successes". This is something that could be optimized in the future, possibly with a different resampling technique.

Mapping Earthquakes with JavaScript and APIs

The purpose of this project is to visually show the differences between the magnitudes of earthquakes all over the world for the last seven days. To illustrate the severity of earthquakes in relation to tectonic plates, an API call to the tectonic plate data is made using D3, and the data is added as an overlay to the map using the GeoJSON layer.

Each earthquake is visually represented by a circle and color, where a higher magnitude will have a larger diameter. JavaScript ES6 was used to create functions to add GeoJSON data, features and interactivity to the maps.

EDA and Matplotlib

The project consisted of providing insights for a fictitious Python-based ride-sharing app company, through an exploratory analysis on their data, creating visualizations using Matplotlib to tell a compelling story.

The analysis is done utilizing Pandas, Scipy and Numpy, as well as Matplotlib visualizations, and showcases the relationship between the type of city and the number of drivers and riders, the percentage of total fares, riders and drivers by type of city to help improve access to ride-sharing services and determine affordability for underserved neighborhoods.
